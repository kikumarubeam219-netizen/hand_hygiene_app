# Hand Hygiene 5 Moments Tracker - デザイン計画書

## 概要
医療従事者向けの手指衛生記録アプリ。WHOの5つのタイミングをイラストでタッチして記録し、統計情報を表示します。

## スクリーン一覧

### 1. ホーム画面（メイン記録画面）
5つのタイミングをカラフルなカード/ボタンで表示。各タイミングをタッチして記録フローに進む。

**表示内容:**
- 5つのタイミングのイラスト/アイコン
- 各タイミングの名前と簡潔な説明
- 本日の実施数カウンター

**主要機能:**
- タイミング選択
- 統計画面へのナビゲーション
- チェックシート画面へのナビゲーション

### 2. 記録画面（モーダル/シート）
選択したタイミングに対して、実施内容を記録する画面。

**表示内容:**
- 選択されたタイミング名と説明
- 実施内容の選択肢：
  - ☐ 手指消毒
  - ☐ 手洗い
  - ☐ 実施なし
- 時刻（自動入力）
- 備考欄（オプション）

**主要機能:**
- 実施内容の選択
- 記録の保存
- キャンセル

### 3. 統計画面
日別、週別、月別の統計情報を表示。

**表示内容:**
- 期間選択（日/週/月）
- 5つのタイミング別の実施数
- 実施率（パーセンテージ）
- グラフ/チャート表示
- 合計実施数

**主要機能:**
- 期間フィルター
- データのエクスポート（オプション）

### 4. チェックシート画面
添付PDF資料のような形式でのチェックシート表示。

**表示内容:**
- 日付/セッション情報
- 5つのタイミング × 複数行（記録）
- 各記録の実施内容チェック
- 実施者情報（オプション）

**主要機能:**
- スクロール表示
- 記録の編集
- チェックシートのダウンロード/PDF出力

### 5. 設定画面
ユーザー設定と統計データの管理。

**表示内容:**
- ユーザー名/施設名入力
- データリセット
- 情報

**主要機能:**
- 設定の保存
- データの初期化

## 主要ユーザーフロー

### フロー1: 新規記録
1. ホーム画面で5つのタイミングのいずれかをタッチ
2. 記録画面が表示される
3. 実施内容を選択（手指消毒/手洗い/実施なし）
4. 保存ボタンをタッチ
5. ホーム画面に戻り、カウンターが更新される

### フロー2: 統計確認
1. ホーム画面の統計ボタンをタッチ
2. 統計画面が表示される
3. 期間を選択（日/週/月）
4. グラフと数値で統計情報を確認

### フロー3: チェックシート確認
1. ホーム画面のチェックシートボタンをタッチ
2. チェックシート画面が表示される
3. スクロールして記録を確認
4. 必要に応じて編集またはダウンロード

## カラースキーム

5つのタイミングを色分けして識別しやすくします。

| タイミング | 色コード | 説明 |
|-----------|--------|------|
| 1. 患者に触れる前 | #FF6B6B | 赤系 |
| 2. 清潔／無菌操作の前 | #4ECDC4 | 青緑系 |
| 3. 体液に曝露された可能性のある場合 | #FFE66D | 黄色系 |
| 4. 患者に触れた後 | #95E1D3 | 緑系 |
| 5. 患者周辺の物品に触れた後 | #C7CEEA | 紫系 |

**基本カラー:**
- 背景：#FFFFFF（ライト）/ #151718（ダーク）
- テキスト：#11181C（ライト）/ #ECEDEE（ダーク）
- アクセント：#007AFF（ブルー）

## レイアウト・デザイン原則

### ポートレート方向（9:16）での最適化
- 画面高さを活用して、5つのタイミングを2行+1行で配置
- または、スクロール可能なグリッドで表示

### ワンハンド操作対応
- 主要ボタンは画面下部1/3に配置
- タッチターゲット：最小44pt
- 親指で操作しやすい配置

### タイポグラフィ
- タイトル：32pt、太字
- サブタイトル：20pt、太字
- ボディ：16pt、通常
- キャプション：12pt、通常

### スペーシング
- 8ptグリッド基準
- 主要間隔：8, 12, 16, 24, 32pt

### コーナー半径
- ボタン：8-12pt
- カード：12-16pt
- モーダル：16-24pt

## アイコン・イラスト

5つのタイミングのイラストは、医療現場での実際のシーンを簡潔に表現します。

| タイミング | イラスト概要 |
|-----------|-----------|
| 1. 患者に触れる前 | 医療従事者が患者に接近する場面 |
| 2. 清潔／無菌操作の前 | 医療用具や注射器を準備する場面 |
| 3. 体液に曝露された可能性のある場合 | 体液に接触した場面（採血など） |
| 4. 患者に触れた後 | 患者から手を離す場面 |
| 5. 患者周辺の物品に触れた後 | ベッドや医療機器に触れた後の場面 |

## ナビゲーション

タブバー（底部）で主要画面を切り替え：
- **ホーム**：記録画面
- **統計**：統計画面
- **チェックシート**：チェックシート画面
- **設定**：設定画面

## データ保存

- **ローカルストレージ**：AsyncStorageを使用して、記録データをデバイス内に保存
- **構造**：JSON形式で、タイムスタンプ、タイミングID、実施内容を記録

## アクセシビリティ

- 高コントラスト色の使用
- テキストサイズの調整可能性
- スクリーンリーダー対応
- 大きなタッチターゲット（44pt以上）

## パフォーマンス

- FlatListを使用してリスト表示を最適化
- 統計データはメモ化して不要な再計算を防止
- 画像はローカルアセットとして最適化
